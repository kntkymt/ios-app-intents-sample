# Overview

This repository contains sample projects for AppIntents

# Samples

- [visual-intelligence-process](./visual-intelligence-process/): A sample project for [Visual Intelligence](https://developer.apple.com/documentation/VisualIntelligence) , demostrating the Visual Intelligence runs on app's process 
- [multi-targets-shared-intent-process](./multi-targets-shared-intent-process/): A sample project demonstrating the process execution behavior of shared App Intents across multiple targets.
- [bazel-visual-intelligence-intent](./bazel-visual-intelligence-intent/): A sample project for [Visual Intelligence](https://developer.apple.com/documentation/VisualIntelligence) with bazel build system.
- [bazel-multi-app-intents](./bazel-multi-app-intents/): A sample project for using multiple app intents modules on ios_application.
